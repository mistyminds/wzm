# 纳米气相分子泵交互工作台

This repository contains a static, graphics-heavy explainer for the carbon nanocone / nanotube gas-phase molecular pump concept.

Open `index.html` directly in a browser, or serve the repository root with any static file server.

## What is interactive

- Particle animation for double-nanocone and eccentric cone-tube structures
- Controls for temperature, interlayer gap, saddle barrier, and density / chemical-potential bias
- Live forward, backward, net crossing, and approximate z-score indicators
- Effective free-energy plot
- Detailed-balance proof diagram
- Event-significance chart using the current manuscript statistics
- Effective-path vs local-recrossing visual classifier

## Scientific scope

The site presents an explanatory model. It does not replace long-time LAMMPS simulations, force-field validation, multiple random seeds, or strict path classification.
